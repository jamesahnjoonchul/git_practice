HTML Basics

HTML stands for Hyper Text Markup Language

  • Most HTML elements contain opening and closing tags with raw text or other HTML tags between them
  • HTML elements can be nested inside other elements. The enclosed element is the child of the enclosing parent element.
  • Any visible content should be placed within the opening and closing <body> tags
  • Headings and sub-headings, h1 to h6 tags, are used to enlarge text. p, span and div tags specify text or blocks. The em and strong tags are used to emphasize text. Line breaks are created with the br tag.
  • Don't forget to use !DOCType html to specify that it is a HTML file
  • open and close with <html>
  • you can link HTML file to other linkes thorugh a anchor. Use href to copy link. You can also wrap img with a element to make img a link
  • use id= to link to a target

How to use tables

Notation Description
< table > "table, always start with this"
< thead > "table head, section off table head"
< tbody > "table body, section off table body"
< tr > "table row, create a new row"
< td > "table data, insert table data"
< tfoot > "table footer, section off table foor for final calculation or summary"

How to use style and use CSS

You can use in-line styling, syntax is <p style="color: red; font-size: 20px" >


But you can write CSS code within HTML

Notation Description
<style> Must be placed within <head> section, write CSS code here
<link> Must be placed within <head> section, used to link to separate CSS file. Must include href, type (="text/css"), rel (="stylesheet")

Using CSS

  • Use a tag name to select elements eg p for paragraph
  • a period (.) must be prepended to the class’s name
  • To select an id element, CSS prepends the id name with a hashtag (#)
  • combining multiple selectors, which we will refer to as chaining. For instance, if there was a .special class for h1 elements
  • or remove repetition by adding a comma
